# 더 워케이션 — MVP 기능 목적-기능 매칭표

> 작성일: 2026-05-06
> 기준: 사업계획서 3대 핵심 기능 + B2B2G 구조 기반
> 우선순위: ★ MVP 필수 / ◎ CBT(3~6개월) / ○ 정식 출시 이후
