## 1. 사용자 4명

| 코드 | 사용자 | 하는 일 | 가장 큰 불편 |
|:---:|---|---|---|
| **HR** | 기업 인사·총무 담당자 | 워케이션 기획, 예산 집행, 정산 처리 | 영수증 150장 수기 대조, 지원금 받으려면 서류 지옥 |
| **EMP** | 직원 (워케이션 참여자) | 숙소 찾기, 예약, 체류 | 숙소 선택권 없음, 신청하면 눈치 보임 |
| **GOV** | 지자체 담당자 | 지원금 예산 등록, 집행 실적 확인 | 기업 데려올 채널이 없어서 예산 못 씀 |
| **ADM** | 더워케이션 운영팀 | 숙소 관리, 회원 관리, 정산, 고객 지원 | 숙소 확보, 기업·지자체 연결, 수수료 정산 |

## 2. MVP 필수 기능 — ★ (0~3개월)

> 이것만 있어도 기업이 워케이션을 예약하고, 비용을 미리 확인하고, 현황을 관리할 수 있다.

### 👤 인사·총무 담당자 (HR)

| # | 기능 이름 | 쉬운 설명 |
|:---:|---|---|
| 1 | 회사 계정 만들기 | 회사 이름·사업자번호·규모 등록해서 플랫폼 가입 |
| 2 | 직원 한 번에 초대하기 | 이메일 목록으로 직원 일괄 초대, 부서별 권한 설정 |
| 3 | 단체 예약 | 날짜·인원·숙소를 한 번에 지정해서 예약 |
| 4 | 예약 현황 보기 | 우리 회사 직원들 예약 상태 전체 목록 확인 |
| 5 | 지원금 적용 후 비용 미리 보기 | 지자체 지원금 빼고 우리가 실제 내야 할 금액 미리 계산 |
| 6 | 전체 현황판 | 예약·비용·지원금 사용 내역 한 화면에서 확인 |
| 7 | 예산 사용률 확인 | 부서별·기간별 예산 대비 얼마나 썼는지 확인 |

| 10 | 숙소 상세 페이지 | 사진·편의시설·와이파이·책상 환경·후기 확인 |

## 3. 시범 운영 기능 — ◎ (3~6개월)

> 초기 기업 3~5곳 확보 후, 정산 자동화와 지자체 지원금 연동으로 핵심 차별점을 완성한다.

| 20 | 결제할 때 지원금 즉시 할인 | 결제창에서 지자체 지원금이 바로 빠지고 나머지 금액만 결제 |

## 5. 사용자별 사용 흐름

### 인사·총무 담당자 (HR)
```
회사 계정 만들기 → 직원 초대
        ↓
숙소 찾기 또는 단체 예약 설정
        ↓
지원금 빼고 실제 비용 미리 확인
        ↓
현황판에서 예약·예산·지원금 한눈에 관리
```

### 직원 (EMP)
```
초대 링크로 가입
        ↓
지도에서 숙소 찾기 (업무 인증 마크 확인)
        ↓
원하는 숙소 직접 예약
        ↓
출발 전 알림 수신 → 체류 중 영수증 사진 제출
```

### 지자체 담당자 (GOV) — 시범운영 단계부터
```
지자체 전용 계정 만들기
        ↓
지원금 예산 코드·한도·조건 등록
        ↓
기업들이 예약할 때 지원금 자동 차감
        ↓
월별 집행 실적 자동 리포트 수신
```

### 더워케이션 운영팀 (ADM)
```
제휴 숙소 등록 및 품질 심사
        ↓
예약 들어오면 알림 수신 → 숙소 파트너 전달
        ↓
기업·지자체 회원 관리 + 문의 처리
        ↓
월별 정산 처리 + 수수료 확인
```
