# 사전 질문지 답변 해석 가이드 (인터뷰어 전용)

> 인터뷰 전날, 상대방 답변을 받은 후 이 가이드로 인터뷰 방향을 세팅한다.

---

## Q1 답변 → 인터뷰 톤 조정

| 답변 패턴 | 인터뷰에서 할 것 |
|---|---|
| 10~30명 | CEO 동행 가능성 높음. Q1에서 "대표님도 같이 가셨나요?" 추가 |
| 제조·현장직 포함 업종 | 직군 형평성 불만 가능. Q10(팀 분위기) 탐색 시 현장직 언급 확인 |
| IT·마케팅 등 사무직 전용 | 형평성 이슈 낮음. 업무 환경·생산성 쪽으로 집중 |

---

## Q2 답변 → Q7 분기 방향 사전 결정

| 답변 패턴 | 인터뷰에서 할 것 |
|---|---|
| 1회 경험 | 섹션 3(재도입 장벽)에 시간 집중. "왜 또 안 가셨어요?"가 핵심 |
| 2회 이상 | "두 번째 장소는 어떻게 찾으셨어요?"(Q8) 심화 탐색 |
| 6개월 이내 | 기억이 선명 → 디테일 캐낼 수 있음. 섹션 1을 빠르게 지나가도 됨 |
| 1년 이상 전 | 기억 흐릿할 수 있음 → 섹션 1에 3~5분 더 쓰고 기억 되살린 후 진행 |

---

## Q3 답변 → 세일즈 진입점 파악

| 답변 패턴 | 인터뷰에서 할 것 |
|---|---|
| 대표 제안 | "대표님도 같이 가셨나요?" 확인. CEO가 직접 체험한 케이스면 CEO용 기능 수요 탐색 |
| 담당자 제안 | Q11(CEO 보고 방식) 심화. 담당자가 설득 도구를 얼마나 원하는지 확인 |
| 팀원 요청 | 직원 수용도(Q10)는 이미 높음. 기획 부담(Q9)에 집중 |

---

## Q4 답변 → Q5 시작 방향 결정 (H2 분기)

| 답변 패턴 | 인터뷰에서 할 것 |
|---|---|
| 받았음 | H2 반증. "신청 과정이 어떠셨어요? 쉬우셨나요?"로 디테일 탐색 |
| 알았지만 안 함 | H2 직접 검증. 이유란에 적힌 내용을 Q5 첫 문장으로 바로 꺼내기 |
| 몰랐음 | H2 강하게 검증. Q5에서 "금액을 미리 알았다면 결정이 달라졌을 것 같으세요?" 추가 |

---

## Q5 답변 → 인터뷰 첫 문장 재료 ★가장 중요

**이 답변으로 인터뷰를 시작한다.**

> "사전에 [Q5 내용]이 불편하셨다고 적어주셨는데, 그때 어떤 상황이었는지 조금 더 얘기해주실 수 있어요?"

이 한 마디로 "이 사람이 내 얘기 읽고 준비했구나"를 느끼게 되면서 대화가 빠르게 열린다.

| 답변 패턴 | 인터뷰에서 집중할 섹션 |
|---|---|
| 와이파이·장소 품질 | Q2(장소 발견) → Q8(재탐색) → Q13(인증 숙소) 중심 재편 |
| 정산·비용 처리 | Q4(비용 처리) → Q5(지원금) → Q6(재도입 비용) 중심 재편 |
| 기획·일정 조율 | Q9(기획 담당 구조) 심화. 기획 자동화 기능 수요 확인 |
| 팀원 불만·불참 | Q10(팀 분위기) 심화. 직원 수용도 낮은 이유 파악 |

---

## Q6 답변 → 섹션 5 비중 조정

| 답변 패턴 | 인터뷰에서 할 것 |
|---|---|
| 날짜 잡혀 있음 | 섹션 5 솔루션 탐색을 앞당겨 진행. 실구매 전환 가능성 높으므로 Q14에서 데모 미팅 제안까지 목표 |
| 하고 싶지만 계획 없음 | H1 탐색 최적 대상. Q7·Q8·Q9에 시간 집중. "무엇이 해결되면 바로 갈 것 같으세요?" 추가 |
| 계획 없음 | 섹션 3 재도입 장벽 먼저 깊게 파고든 후 솔루션 반응 확인. Q13·Q14에서 "이게 있었다면 달라졌을까요?" 강하게 활용 |

---

## 인터뷰 전날 체크리스트

```
□ Q1 → 직원 수·업종 파악, 인터뷰 톤 결정
□ Q2 → 횟수 확인, Q7 분기 방향 머릿속에 세팅
□ Q3 → 결정 주체 확인, 세일즈 진입점 파악
□ Q4 → 지원금 인지 여부 확인, Q5 시작 방향 결정
□ Q5 → 불편했던 점 읽고 인터뷰 첫 문장 준비
□ Q6 → 다음 계획 여부, 섹션 5 비중 결정
```
