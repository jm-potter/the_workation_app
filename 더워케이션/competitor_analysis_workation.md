### 1-1. 디어먼데이 (Dear Monday)

| 항목 | 내용 |
|------|------|
| **공식 URL** | https://dearmonday.io / https://dear-monday.co.kr |
| **설립연도** | 2022년 10월 |
| **대표** | 권유진 (전 대기업 인사담당자) |
| **타겟** | B2B (기업 인사/조직문화 담당자) |
| **슬로건** | "월요병 없는 세상" |

#### 투자 유치 현황
- **프리시드 (2022.10)**: 앤틀러코리아 → **2억 원**
- **시드 (2025.02)**: 롯데벤처스 + VNTG + 앤틀러코리아 → 금액 미공개 (추정 5~10억원 규모)
- 한국관광공사 사장상 수상
- 앤틀러 코리아 1기 배치 프로그램 출신

#### 핵심 서비스
- 전국 프리미엄 워케이션 지점 운영 (제주 롯데호텔점, 경주 춘추관점, 강릉 세인트존스호텔 등)
- **국내 유일 워케이션 특화 예약 시스템**: 기업 인사담당자가 직원 예약 → 이용현황 파악 → 간편 정산까지 원스톱 처리
- 업무 환경 풀세팅 (하이엔드 의자, 모니터, 전용 회의실) → 노트북만 가져오면 OK
- B2G 협력: 경북형 워케이션 지원금 대행 (경주·안동·영천 지점)

#### 가격 정책
- 공개 가격 없음 (기업별 문의 기반 B2B 계약)
- 경북형 워케이션 지원금 연계 상품: 숙소 할인 + 교통비 최대 3만원 환급 형태로 운영
- 지자체 지원금 적용 시 기업 실부담 대폭 감소

#### 기능 상세 평가

| 기능 | 제공 여부 | 비고 |
|------|----------|------|
| 정부 지원금 행정처리 대행 | 부분 제공 | 경북형 지원금은 플랫폼 내 처리, 타 지자체는 미확인 |
| 업무 성과 보고서 | 미확인 | 공개 자료 없음 |
| 직원 숙소 자율선택 | 미확인 | 지정 지점 내에서 선택 가능 추정 |
| 단일 청구서 정산 | 부분 제공 | 간편 정산 언급, 세금계산서 통합 여부 미확인 |
| 대시보드 (이용현황) | 제공 | HR 담당자용 이용현황 파악 기능 보유 |

#### 차별점
- 숙소 운영 노하우 + SaaS형 예약관리 결합
- 프리미엄 호텔(5성급 롯데호텔 등)과 파트너십 → 퀄리티 보증
- 지역 관광재단·지자체와 직접 협업 → B2G 레퍼런스 보유

#### 약점 (TheWorkation 관점)
- **정산 자동화 미완성**: 수십 명 영수증 수기 대조 문제 미해결
- **지자체 연동 한계**: 경북 일부만 지원금 연동, 전국 B2G 네트워크 부재
- **숙소 자율선택 제한**: 지정 지점만 이용 가능 → 직원 선택권 협소
- 공개된 ESG/업무성과 보고서 기능 없음

### 1-2. 스트리밍하우스 / 더휴일 (THE HYUIL)

| 항목 | 내용 |
|------|------|
| **공식 URL** | https://thehyuil.co.kr |
| **설립연도** | 2020년 6월 |
| **대표** | 신동훈 |
| **타겟** | B2B + B2G (기업 + 지자체) |
| **슬로건** | "일하며 살아가는 매일을 더 특별하게" |

#### 투자 유치 현황
- 한국관광공사 관광벤처(성장) 선정 (2024.05)
- VISIT KOREA 얼라이언스 선정 (2024.04)
- 한국어촌어항공단 어촌체험휴양마을 워케이션 운영 용역 (2024.03)
- 스크럼벤처스 HFX 글로벌 액셀러레이팅 프로그램 선정
- 미네르바대학교 MOU 체결 (글로벌 런케이션)
- 드레이퍼 스타트업 하우스 코리아와 파트너십
- 구체적 투자금액은 미공개

#### 핵심 서비스
- 워케이션 솔루션 원스톱 제공: 지역 기획·개발 → 기업 컨설팅 → 코디네이터 → 공유오피스/숙소/로컬체험 결합 → 예약·일정 조율 → 현장 서비스 → 사후관리 → **워케이션 레포트** 제공
- **2,000여 개 기업 + 50여 개 지자체** 파트너십 보유 (국내 최다 수준)
- 한국관광공사 2025 워케이션 지원 프로그램의 공식 운영 플랫폼 (신청처: thehyuil.co.kr)
- 부산형 워케이션 거점센터 위성센터 운영 참여 (더휴일부산)

#### 가격 정책
- 기업별 맞춤 견적 (B2B 계약)
- 한국관광공사 지원 프로그램과 연계: 1인당 최대 10만원 지원

#### 기능 상세 평가

| 기능 | 제공 여부 | 비고 |
|------|----------|------|
| 정부 지원금 행정처리 대행 | **제공** | 한국관광공사 공식 신청창구 역할 |
| 업무 성과 보고서 | **제공** | "워케이션 레포트" 명시 |
| 직원 숙소 자율선택 | 부분 제공 | 제휴 숙소 중 선택 가능 |
| 단일 청구서 정산 | 미확인 | 자동화 여부 불명확 |
| 대시보드 | 미확인 | |

#### 차별점
- **업계 최다 지자체 연동 (50개+)** → B2G 레퍼런스 압도적
- 워케이션 레포트 제공 → 기업 복지 보고 니즈 충족
- 한국관광공사 공식 플랫폼 지위 → 신뢰도·인지도 최상위

#### 약점 (TheWorkation 관점)
- **정산 자동화 부재**: 지원금 수령은 안내하지만 영수증 대조·정산 자동화는 별도 서비스
- **가격 투명성 부재**: 공개 요금제 없음
- **SaaS 성격 약함**: 인력 중심 서비스 (코디네이터 투입) → 확장성 한계
- **숙소 자율선택 제한**: 지정 파트너사 중심

### 1-3. 달램 (Dallem)

| 항목 | 내용 |
|------|------|
| **공식 URL** | https://dallem.com |
| **설립연도** | 미공개 (2019~2020년 추정) |
| **타겟** | B2B (기업 임직원 웰니스) |
| **핵심 포지셔닝** | EAP(근로자지원프로그램) + 웰니스 통합 플랫폼 |

#### 투자 유치 현황
- 구체적 투자 라운드 미공개
- 서울창조경제혁신센터 등 다수 기업 고객 보유

#### 핵심 서비스
- 워케이션은 부가 서비스 (핵심은 EAP + 웰니스)
- 강원도 정선 '파크 로쉬'와 단독 계약 → 웰니스 워케이션 프로그램
- 오피스 스트레칭, 근골격계 예방, 직무 스트레스 관리, 심리상담, 웰니스 특강 결합
- 싱잉볼 테라피, 그림책 테라피, 다도 등 힐링 액티비티 포함
- 사무실로 직접 찾아가는 맞춤형 웰니스 솔루션

#### 가격 정책
- 기업별 맞춤 계약 (구독형 EAP + 워케이션 패키지 번들)

#### 기능 상세 평가

| 기능 | 제공 여부 | 비고 |
|------|----------|------|
| 정부 지원금 행정처리 대행 | 미제공 | 복지 서비스 중심, 지원금 연동 미확인 |
| 업무 성과 보고서 | 미확인 | EAP 리포트는 제공 가능 추정 |
| 직원 숙소 자율선택 | 미제공 | 지정 파트너 시설만 이용 |
| 단일 청구서 정산 | 미확인 | |

#### 차별점
- 심리/웰니스 전문성 → 단순 숙박이 아닌 직원 건강 케어
- EAP 시장(기업 복지 예산)과 워케이션 결합 → 별도 예산 확보 용이

#### 약점 (TheWorkation 관점)
- 워케이션이 메인 사업이 아님 → 전문성·규모 모두 제한
- 지원금 행정 기능 없음
- 숙소 선택 폭 매우 좁음 (정선 1곳 단독계약)

#### 기능 상세 평가

| 기능 | 제공 여부 | 비고 |

#### 차별점
- 프리랜서/개인/소규모팀 특화 → 틈새시장 공략

#### 약점 (TheWorkation 관점)

- 프라이머 배치 19기 선정 (프리시드)

- 단기: 일 단위 코워킹
- 장기: 월 단위 거주+업무 패키지

## 5. TheWorkation 포지셔닝 시사점

### 핵심 발견사항

**1. 정산 자동화 공백 확인**
- 현존 한국 1위 더휴일조차 "워케이션 레포트"는 제공하나 **1-Billing 정산 자동화는 없음**
- 디어먼데이는 간편정산 언급하나 세금계산서 통합 자동화 미확인
- 모든 경쟁사에서 "영수증 150장 → 1장 통합" 기능 부재 → **TheWorkation의 가장 강력한 해자**

**2. B2G 진입장벽 현실 확인**
- 더휴일이 50개 지자체 연동으로 선점 → 그러나 **결제창 실시간 차감 (B2G 결제망)** 은 아무도 구현 못함
- 현재 모두 "사후 지원금 환급" 방식 → 기업 선결제 후 대기 불편 지속
- **TheWorkation의 "선공제(즉시 할인)" 모델** → 경쟁사 전체와 차별화

**3. 숙소 자율선택 공백**
- 디어먼데이: 지정 지점만 (제주 롯데호텔 등)
- 더휴일: 제휴 숙소 중 선택
- **직원 50명이 각자 원하는 숙소를 고르면서 한 번에 정산되는 기능** → 시장에 없음

**4. 글로벌 공간 중심 모델의 실패**
- Selina ($198M 연간 손실) + Remote Year 폐업 → **공간 소유·임대 기반 모델의 취약성 입증**
- SaaS/핀테크 레이어로 시작하는 **에셋라이트 모델** 유효성 확인

**5. 시장 성장 확실성**
- 제주: 2024년 9,760명 → 2026년 목표 10만명 (10배 성장 목표)
- 충남: 전년 대비 1.7배 증가 (2025년)
- 강원: 주중 체류 40% 증가 (2024년)
- 직장인 90% 워케이션 희망, 기업 도입률 19.9% → **갭이 곧 시장**

### TheWorkation 경쟁우위 요약

```
경쟁사가 하는 것:        → TheWorkation이 더하는 것:
──────────────────────────────────────────────────
예약 중개              → B2G 결제망 (선공제 즉시 할인)
간편 정산              → 1-Billing 세금계산서 1장 자동화
워케이션 레포트         → ESG 임팩트 리포트 (대기업 지속가능경영 연동)
지정 숙소              → 직원 자율선택 + 통합 법인결제
지자체 안내            → 지자체 예산 실시간 차감 결제망
```
