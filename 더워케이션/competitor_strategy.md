## 1. 경쟁사별 페인포인트 해결 방식 및 성과

### 디어먼데이 (Dear Monday)

| 항목 | 내용 |
|---|---|
| **설립** | 2022년 10월 / 직원 5명 |
| **모델** | B2B SaaS |
| **해결 방식** | 기업 HR 담당자 전용 예약 시스템. 직원별 예약·이용 현황·기본 정산을 하나의 화면에서 처리. 숙소 테마별 큐레이션(5성급 호텔~한옥~반려동물 동반) |
| **성과** | 2024년 매출 5억 8,186만원 / 롯데벤처스·VNTG 시드 투자 유치 |
| **미해결 페인포인트** | 지원금 자동화 없음 / 영수증 정산 수동 / 성과 리포트 없음 / 선공제 없음 |

| 항목 | 내용 |
|---|---|
| **설립** | 2020년 6월 |
| **모델** | B2G 중심 + B2B·B2C 병행 |

## 4. 시장 점유 전략

### 전략 구조: 3단계 점진적 포위

```
1단계 (0~1년)     2단계 (1~3년)     3단계 (3년+)
디어먼데이 고객   더휴일 고객 흡수   현대이지웰·야놀자
겨냥 → 초기       → 전국 확장        공략 → 대기업
레퍼런스 확보                         ESG 세일즈
```

---

### 1단계 — 디어먼데이 타겟 공략 (0~1년차)

**왜 디어먼데이를 먼저 타겟으로 삼는가**
- 매출 5.8억원, 팀 5명 → 서비스 확장 속도 느림
- 선공제·1-Billing·성과 리포트 미제공 → 고객 불만 존재
- 롯데벤처스 투자로 대기업 채널 생기기 전에 선점 필요

**실행 방법**
- 디어먼데이 고객사(HR 담당자)가 모이는 채널(리멤버 HR 그룹, LinkedIn)에서 "영수증 150장 → 청구서 1장" 메시지로 직접 접근
- 디어먼데이 대비 3가지 추가 기능(선공제·1-Billing·성과 리포트)을 명시한 비교 랜딩 페이지 운영
- 무료 파일럿 10개사 → 레퍼런스 확보 → 인바운드 세일즈 전환

**KPI**: 파트너 기업 5개사, GMV 12억원, 정산 자동화 오류율 0%

---

### 2단계 — 더휴일 고객 흡수 (1~3년차)

**왜 더휴일이 흡수 가능한가**
- B2G 의존 구조 → 정부 예산 삭감 시 취약
- 코디네이터 인력 기반 → 자동화로 대체 불가 수준의 마진 압박
- AI 자동화 없음 → 대기업이 ESG 리포트 요구하면 대응 불가

**실행 방법**
- 더휴일이 MOU 맺은 지자체(50개) 중 집행률 낮은 곳을 역으로 공략
- "더휴일은 코디네이터가 해드리는 것, 더 워케이션은 시스템이 알아서 하는 것" 메시지
- 대기업 ESG 담당자 직접 접촉 → ESG 임팩트 리포트 자동 생성 기능으로 결재라인 공략

**KPI**: 지자체 10개+ B2G 연동, 엔터프라이즈 20곳, GMV 50억원

---

### 3단계 — 현대이지웰·야놀자 대응 (3년+)

**왜 직접 충돌을 피해야 하는가**
- 현대이지웰: 거래액 8,428억원, 고객사 2,500개, 임직원 300만명 → 정면 충돌 불가
- 야놀자: 매출 9,245억원, 기술력 보유 → 워케이션 진출 시 강력한 경쟁자

**차별화 전략: 틈새 → 표준**
- 현대이지웰이 할 수 없는 것: 지자체 B2G 직접 연동, AI 성과 리포트, 근태 증빙 자동화
- 야놀자가 들어오기 전에 지자체 MOU 네트워크 선점 → 진입 장벽 구축
- 더 워케이션의 정산 데이터가 쌓일수록 대기업도 이탈 불가한 락인 구조 완성

**KPI**: GMV 150억원, SaaS 구독 전환 30%, 글로벌 진출

## 5. 고객 유형별 세일즈 메시지

### HR·재무 담당자

> "디어먼데이는 예약을 잡아드립니다. 더 워케이션은 영수증 150장을 지웁니다."

- 7일 수기 정산 → 당일 자동화
- 지원금 사후 환급 → 결제 시 즉시 차감
- 경영진 결재 안 남 → AI 성과 리포트로 근거 자동 생성

### 지자체 담당자

> "더휴일은 기업을 데려다 드립니다. 더 워케이션은 예산이 자동으로 집행됩니다."

- MOU 기반 예산 코드 등록 → 기업 결제 시 자동 차감
- 월 집행 내역 자동 정산 → 행정 인력 0명으로 운영 가능
- 소비 데이터 자동 집계 → 의회 예산 심의 자료 자동 생성

### 임직원

> "회사가 지정한 숙소 말고, 내가 원하는 곳으로."

- 예산 범위 내 자율 선택
- 업무 환경 인증 필터 (Wi-Fi·책상 보장)
- 간편 신청·자동 승인으로 눈치 없이 신청

## 6. 시장 진입 타임라인

| 시기 | 타겟 | 핵심 액션 | 목표 |
|---|---|---|---|
| **0~3개월** | 수도권 IT 스타트업 | 무료 파일럿 10개사, 강원 지자체 MOU 1건 | 레퍼런스·데이터 확보 |

| **6~12개월** | 더휴일 지자체 파트너 역공략 | 집행률 낮은 지자체 직접 영업, AI 성과 리포트 런칭 | 지자체 MOU 5개, GMV 12억 |

| **2~3년차** | 현대이지웰 고객 침투 | 복지포인트 연동 API 제공, 기업복지 플랫폼 화이트레이블 | GMV 150억, AI SaaS 구독화 |
| **3년+** | 글로벌 (일본 우선) | Stripe 연동, 영문 리포트, 일본 지자체 파트너십 | 해외 GMV 추가, PCT 특허 활용 |
