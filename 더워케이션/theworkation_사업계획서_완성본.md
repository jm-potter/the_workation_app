# 더 워케이션 (TheWorkation)
## 2027 예비창업패키지 사업계획서 (AI 분야)

---

## ▣ 0. 사업 아이템 개요

**팀명:** 더 워케이션 (TheWorkation)

**아이템 한 줄:** AI 기반 성과·ESG 리포트와 자동 정산으로 기업의 워케이션 도입 장벽을 제거해, 8,190억원의 지방소멸대응기금을 실제 지방 경제로 연결하는 B2B SaaS

**창업 목표:** 막혀 있는 지방소멸대응기금을 기업의 워케이션으로 흐르게 하고, 이 구조를 글로벌로 확장하는 워케이션 인프라 기업으로 성장

## ▣ 1. 창업 배경 및 개발동기

> - 캡처 1순위: [kr.vizney.com — 지방소멸 위험 지도 2025](https://kr.vizney.com/regional-extinction-risk-map-of-south-korea-2025/)
> - 캡처 2순위: [한국고용정보원 공식 자료](https://www.keis.or.kr/keis/ko/bbs/145/detail.do?pstSn=63579)

---

- 전국 228개 시군구 중 **118개(51.8%)가 소멸위험지역**. 정부는 2022~2031년 연 1조원, 총 10조원 규모의 지방소멸대응기금을 편성하고 '워케이션을 통한 생활인구 유입'을 핵심 수단으로 지정함.
    - 출처: 행정안전부, 지방소멸대응기금 투자계획 (2022)
- 그러나 2024년 기준 기금 집행률은 **18.1%** — **약 8,190억원이 방치**됨. 직장인 90%가 워케이션을 원하지만 실제 도입률은 **19.9%**에 그침.
    - 출처: 나라살림연구소·행정안전부 (2024) / 대한상공회의소 (2023, n=1,112)
- 70%p의 공백은 수요 부재가 아님. **기업에 도입 이유가 없기 때문임** — 워케이션이 생산성에 기여하는지, 비용 처리는 어떻게 되는지, 아무도 기업에 데이터로 답하지 않았음.
- 호텔경영학 전공·창업학 복수전공, 정부 정책 실시간 모니터링을 통해 이 구조적 공백을 포착하고 AI 기반 B2B SaaS로 구조화함.

## ▣ 2. 창업아이템의 목적 및 필요성

> **[이미지 ②] 90% vs 19.9% 갭 차트** — `style_A_gap.png` 삽입
> - 출처 표기: 대한상공회의소, 「직장인 워케이션 선호도 조사」(2023, n=1,112)

---

1. 아이템의 목적 — 기업이 움직이지 못하는 이유를 AI로 제거

1) AI 성과·ESG 리포트 — 워케이션을 왜 해야 하는지 데이터로 답함
(1) AI 성과 리포트: 협업 툴(Slack·Notion·Google Workspace) API 연동 → 워케이션 기간 생산성·협업 데이터 자동 수집 → LLM이 분석해 내부 보고용 자연어 리포트 자동 생성.
    - 근거: Microsoft·Nature 공동연구 (2021, 6만명) — 완전 재택 시 부서 간 협업 25% 감소 실증. 워케이션과 재택의 차이를 AI가 데이터로 증명함.
(2) AI ESG 임팩트 리포트: 임직원 소비 데이터 기반으로 AI가 지역 경제 기여도를 자동 계산·시각화 → 지속가능경영보고서에 바로 활용 가능한 형태로 생성.

1) AI 지원금 매칭 엔진 — 결제 시점에 비용 불확실성을 제거
(1) 기업 규모·업종·일정 입력 → AI가 신청 가능한 지자체 보조금 자동 매칭.
(2) 결제창에서 매칭된 지자체 보조금 실시간 차감 → 기업 실질 부담 최대 0원.
(3) 경쟁사 전원이 사후 환급 방식 — AI 선공제 구조는 당사만의 핀테크 해자.

1) AI OCR 1-Billing 정산 — 행정 부담을 시스템으로 대체
(1) 임직원 영수증 이미지 촬영 → AI OCR 자동 인식·분류·집계 → 월 세금계산서 1장 자동 발행.
(2) 재무팀 정산 소요: 기존 7일(영수증 150장 수기 처리) → 당일 자동 처리.

2. 필요성
1) 워케이션 도입 기업 만족도 90%, 재참여 희망률 96% (서울경제진흥원, 2023, n=523). 한 번 움직인 기업은 계속 움직임. 더 워케이션은 처음 움직이는 이유를 AI로 만드는 민간 인프라임.

## ▣ 4. 사업화 전략

> **[이미지 ④] B2B2G 서비스 구조도** — `chart_flow.png` 삽입
> - 구성: 지자체(미집행 예산) → 플랫폼(AI 가치 증명 + 실행 자동화) → 기업(도입 결정) → 임직원(지방 체류) → 지방(생활인구 유입)

---

1. 비즈니스 모델 — B2B2G

| 이해관계자 | 현재의 상태 | 더 워케이션이 바꾸는 것 |
|:---:|:---:|:---|
| **지자체** | 8,190억원 미집행, 워케이션 센터 공실 | AI 기업 매칭으로 예산 소진, 생활인구 실적 확보 |
| **기업** | 도입 이유 없어 의사결정 불가 | AI 성과·ESG 리포트로 가치 가시화, 실행 마찰 제거 |
| **임직원** | 원하지만 실행되지 않는 워케이션 | 90%의 잠재 수요가 현실화 |
| **숙소 파트너** | OCC 50%대, 연 182일 공실 | AI 수요 매칭으로 가동률 회복 |

**B2G 결제 작동 메커니즘**

지자체 행정망은 외부 API 직접 연동이 불가능한 폐쇄 구조임. 이를 우회하는 현실적 플로우는 다음과 같음:

① MOU 체결 후, 지자체 담당자가 플랫폼에 예산 코드와 집행 한도를 직접 등록 → ② 기업이 결제 시 해당 코드를 선택해 보조금 적용 금액 확인 → ③ 플랫폼이 월 단위로 지자체 담당자에게 집행 내역 정산 리포트 발송 → ④ 지자체가 예산 집행 처리 완료.

즉 실시간 API 연동이 아닌 **MOU 기반 수기 코드 등록 + 월정산 대행** 방식으로, 행정망 장벽을 우회하면서도 지자체의 집행 실적(생활인구 유입 건수)을 수치로 제공함. 이 구조는 기술이 아닌 지자체와의 신뢰 관계와 MOU 네트워크가 진입장벽이 됨.

2. AI 기술 구조

1) LLM 기반 리포트 생성: GPT-4o API 또는 Claude API → 협업 데이터 분석 → 자연어 보고서 자동 출력
1) AI OCR 정산: 네이버 클로바 OCR API → 영수증 자동 인식·분류·합산
1) 지원금 매칭 엔진: 지자체 보조금 DB + 기업 프로파일 → Rule-based + AI 추천 레이어

3. 수익 구조

**Pricing Tier**

| 플랜 | 대상 | 가격 | 포함 기능 |
|:---:|:---:|:---:|:---|
| **Starter** | 10~50인 스타트업 | 무료 (거래 수수료 12%) | AI 지원금 매칭, AI OCR 정산, 기본 대시보드 |
| **Growth** | 50~200인 중견기업 | 월 99만원 + 수수료 8% | Starter + AI 성과·ESG 리포트, 부서별 예산 분할 |
| **Enterprise** | 200인+ 대기업 | 별도 협의 | Growth + ESG 보고서 커스텀, B2G 직접 연동, SLA 보장 |

**Unit Economics (LAM 기준 — 수수료 단독)**

- 평균 고객사 1개사 연간 GMV: 14명 × 60만원 = **840만원**
- Starter 기준 수수료 수익: 840만원 × 12% = **100만원 / 사**
- 목표 고객사 150개 달성 시 연간 수수료 수익: **약 1.5억원** (1년차 전망)

**구독 전환 시 추가 수익 (2년차 이후 시나리오)**

- 구독 전환율 30% 가정: 45개사 × 99만원 × 12개월 = **약 5.3억원**
- → 수수료 1.5억 + 구독 5.3억 = 합산 약 **6.8억원** (2년차 이후, 구독 전환 안정화 시점 기준)

4. 개발 로드맵

1) 0~3개월: 노코더스(Nocoders) — MVP 구축 (지도API·결제·기본 대시보드)
1) 3~6개월: 그릿지(Gridge) + AI API 연동 — OCR 정산·LLM 리포트 CBT, HR 담당자 10명 피드백 기반 이터레이션
1) 6개월~: 자체 개발 전환 — AI 고도화 (리포트 정확도 개선), B2G 연동, Stripe 연동

## ▣ 6. 시장진입 및 성과창출 전략

**Cold Start 전략 — 데이터 0인 상태에서 첫 고객을 확보하는 방법**

AI 리포트의 정확도는 데이터 누적에 달려 있으나, 첫 고객 유치 시점에는 데이터가 없다. 이 역설을 다음 순서로 해소함:

① **무료 파일럿 (0~2개월)**: 창업보육센터·대학 내 스타트업 커뮤니티를 통해 10개사 직접 섭외 → 비용 0원으로 파일럿 제공, 실사용 데이터 및 레퍼런스 동시 확보.
② **레퍼런스 세일즈 (3~6개월)**: "A사 워케이션 성과 리포트" 사례를 리멤버 HR 그룹·LinkedIn에 공개 → 동일 pain point를 가진 HR 담당자 인바운드 유입.
③ **구독 전환 (6개월~)**: 파일럿 데이터 기반으로 AI 리포트 정확도 입증 → Starter→Growth 플랜 업셀링.

| 단계 | 기간 | 핵심 전략 | KPI |
|:---:|:---:|:---|:---|
| **1단계** 강원 MVP | 0~1년 | 정선·영월·평창 지자체 MOU + HR 핀셋 마케팅 | 제휴 숙소 10곳, 파트너 기업 5곳, GMV 12억 |
| **2단계** 전국 확장 | 1~3년 | 89개 인구감소지역 B2G 연동 + 대기업 ESG 세일즈 | 지자체 10개+, 엔터프라이즈 20곳, GMV 50억 |
| **3단계** 글로벌 인바운드 | 3~5년 | Stripe 연동 + 영문 AI 리포트 → 해외 기업 유입 | 일본 지자체 파트너십 3개 체결, 해외 기업 고객 10사, GMV 300억 |
| **4단계** 글로벌 아웃바운드 | 5년+ | 동아시아·동남아 파트너십 + 원화 단일 정산 | 진출 국가 5개국, GMV 1,000억, 연간 수익 120억 |

## ▣ 8. 창업자의 역량

**"산업 전문성과 창업 실행력을 겸비한 준비된 리더"**

| 구분 | 직위 | 담당 업무 | 보유 역량 |
|:---:|:---:|:---|:---|
| 1 | CEO | 사업 총괄 및 인프라 확보 / 로컬 숙박·오피스 인프라 발굴 및 품질 관리 / 지자체(B2G) MOU 추진 | 호텔경영학·창업학 복수전공 / 다수 창업경진대회 수상 / 창업지원팀·창업보육센터 근로 경험 |

본 사업의 핵심인 '공간(Hotel)'에 대한 실무 지식과 '창업(Startup)'에 대한 실행 경험을 동시에 보유하여 사업의 성패를 좌우할 핵심 역량을 갖춤.

**(1) 호텔 경영 전문성 (Industry Insight)**

호텔경영·창업학 전공자로서 다수의 호텔 인턴십을 통해 숙박 산업의 운영 메커니즘과 고객 서비스 표준을 체득함. 이를 통해 워케이션의 본질인 '체류 공간의 품질'을 전문가적 관점에서 통제할 수 있음. 또한 OCC 50%대 구조에 대한 학술적 이해를 바탕으로 공급과잉 지방 숙소·미집행 예산·기업 수요의 단절 구조를 포착하고, AI 기반 B2B SaaS로 재설계함.

**(2) 창업 실행력 (Entrepreneurship)**

각종 창업경진대회 수상 이력을 통해 아이디어의 비즈니스 모델화, 시장 검증, 자금 조달 등 창업 전 과정에 대한 실전 이해도를 보유함. 창업지원팀·창업보육센터 근로를 통해 초기 스타트업의 실행 구조를 내부자 시각으로 습득함. 이는 초기 스타트업이 겪는 시행착오를 최소화하고 LAM 달성까지 빠르게 도달하는 원동력이 됨.

**(3) 정책 모니터링**

문체부·관광공사·행안부 정책 실시간 추적 → 8,190억원 미집행의 원인을 '기업의 가치 인식 부재'로 진단, AI B2B SaaS로 구조화함.
